   Installation & Configuration
Download

http://webbreaker.io/

Rename

Rename the file to webbreaker or for Windows webbreaker.exe

Environment Settings

Add it to your environment's $PATH or for Windows %PATH%

Pre-Configuration

First time installs, run the webbreaker command in your terminal to create the default configurations, under $HOME/.webbreaker/config.ini

Post-Configuration

Configure the $HOME/.webbreaker/config.ini Webinspect, Fortify SSC, and ThreadFix sections to your environment.

# Your required Fortify SSC URL and run `webbreaker admin credentials --fortify`
[fortify]
ssc_url = https://ssc.example.com/ssc
username =
password =

# Create a ThreadFix API Key from your ThreadFix deployment
[threadfix]
host = https://threadfix.example.com:8443/threadfix
api_key = ENTER API KEY HERE

# Enable WebInspect authentication set it to `true` after it is configured on the server
authenticate = false
username =
password =

That's it, you're done! Happy scanning!
